# SpringBoot在线文档管理系统 java654

## 技术栈

- **后端**: SpringBoot + Mybatis
- **数据库**: MySQL 8
- **前端**: VUE + ElementUI

## 主要开发环境

- MySQL 8
- JDK 1.8
- Maven 3
- IDEA
- Node 14

## 功能模块

### 首页管理

管理员可以在首页进行以下配置和定制：

- 设置系统公告
- 展示最新文档
- 提供快速访问链接
  以确保用户能快速方便地获取所需信息。

### 个人中心

- 管理员可管理用户个人信息：
  - 账户设置
  - 权限分配
  - 密码重置
  确保信息的完整性和安全性。

### 系统登录


### 员工管理

- 管理员可对员工进行管理：
  - 添加新员工
  - 编辑员工信息
  - 调整员工权限
  确保员工信息的完整性和安全性，并有效分配工作任务。

### 文件信息管理

- 管理员可对文档进行管理：
  - 上传新文档
  - 编辑文档内容
  - 设置文档权限
  确保文档的整体管理和安全性，便于用户查找和获取所需文档。
